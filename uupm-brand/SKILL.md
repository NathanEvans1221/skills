# UI UX Pro Max - 品牌指南 (Brand)

當需要「定義品牌調性」、「設定視覺主題」或「撰寫行銷內容」時使用。

## 🖋️ 品牌語調 (Voice & Tone)
*   **權威感 (Authority)**：專業、可靠、精準。
*   **親和力 (Approachability)**：親切、易懂、充滿現代感。
*   **創新性 (Innovation)**：前瞻、極簡、突破常規。

## 👁️ 視覺識別 (Visual Identity)
1.  **標誌使用規範**：維持安全間距，絕對不可拉伸或扭曲 Logo 比例。
2.  **標準色盤**：必須遵循品牌標準色，確保在不同媒介（網站、社群圖、簡報）的一致性。
3.  **字體氣質**：根據產品定位選擇襯線（典雅、權威）或無襯線（現代、科技）字體。

## 💼 資產管理
*   **命名的語義化**：資產檔案名稱應包含類別與用途（例如：`logo_primary_dark.svg`）。
*   **格式選擇**：圖標與標誌優先使用 SVG；情境圖片使用 WebP 以優化載入速度。

## ✅ 品牌一致性檢查
*   [ ] 所有的顏色是否都在定義的品牌色盤內？
*   [ ] 內容的語氣是否符合品牌敘事？
*   [ ] Logo 的使用是否符合比例與留白規則？
