# Presentation Slides Mastery Reference

## 核心簡報結構 (Master Structure)
一份專業的解決方案簡報應包含以下 8-10 頁核心結構：

1.  **標題頁 (Title)**: 震撼的視覺背景 + 核心價值主張。
2.  **痛點分析 (Problem)**: 聚焦於 3 個具體的業務挑戰。
3.  **解決方案 (Solution)**: 高層級的產品架構或核心理念。
4.  **功能亮點 (Key Features)**: 3-5 個關鍵差異化功能。
5.  **產品展示 (Demo/Mockups)**: 實際介面截圖或流程圖。
6.  **商業價值 (Business Value)**: ROI、節省時間、增加收益。
7.  **成功案例/信賴 (Social Proof)**: 客戶證詞、合作夥伴。
8.  **方案/時程 (Roadmap/Pricing)**: 下一步計畫或定價體系。
9.  **行動呼籲 (CTA)**: 明確的 Next Step（例如：預約 Demo）。
10. **問答/聯絡 (Contact)**: QR Code 或聯繫資訊。

## 視覺規範
- **文字量**: 每頁不超過 15 個單詞，核心概念應大於 24px。
- **對比度**: 確保在投影機上也有高對比（避免極細文字）。
- **色彩**: 遵循品牌主色，重點強調色不超過 2 種。
- **動畫**: 僅使用「淡入/淡出」，避免分散注意力的過渡動畫。

## 設計技巧
- **Bento Box Layout**: 適用於功能清單頁。
- **Split Screen**: 適用於「現況 vs. 方案」的對比。
- **Giant Typography**: 適用於關鍵數據或引言。
- **Hero Image Background**: 適用於轉場頁或標題頁。
