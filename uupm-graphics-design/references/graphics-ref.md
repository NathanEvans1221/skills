# Banner & Graphics Quick Reference

## Platform Size Guide
| Platform | Type | Size (px) | Aspect Ratio |
| :--- | :--- | :--- | :--- |
| **Facebook** | Cover | 820 x 312 | ~16:6 |
| **X (Twitter)** | Header | 1500 x 500 | 3:1 |
| **LinkedIn** | Personal | 1584 x 396 | 4:1 |
| **YouTube** | Channel Art | 2560 x 1440 | 16:9 |
| **Instagram** | Story | 1080 x 1920 | 9:16 |
| **Instagram** | Square Post | 1080 x 1080 | 1:1 |
| **Web** | Hero Section | 1920 x 800 | ~21:9 |

## Art Styles for Banners
*   **Minimalist**: 適用於 SaaS、技術文件。
*   **Bold Typography**: 適用於重大公告與活動。
*   **Gradient**: 現代品牌、新創公司。
*   **Photo-Based**: 生活風格、電子商務。
*   **Glassmorphism**: 軟體介面、應用程式功能展示。

## 設計核心規範
1.  **安全區域 (Safe Zones)**：重要內容（文字、Logo）應保持在中央 75% 區域內，避免邊緣被裁切。
2.  **單一 CTA**：每個橫幅應只有一個主要行動呼籲，尺寸不小於 44px 高。
3.  **文字限制**：廣告圖中的文字佔比應低於 20%（遵循 Meta 規範）。
4.  **層級對比**：大標題至少 32px，副標題至少 16px，確保在行動端可見。

## 圖示風格選擇 (Icon Styles)
*   **Outlined (線性)**：Web 介面、後台系統。
*   **Filled (填滿)**：行動 App、導覽底列 (Bottom Nav)。
*   **Duotone (雙色)**：行銷登陸頁、特色說明。
*   **Rounded (圓角)**：親和力強的 App、醫療保健。
*   **Sharp (銳角)**：企業軟體、金融科技。
