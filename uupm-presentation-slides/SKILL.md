# UI UX Pro Max - 簡報投影片 (Slides)

當需要「製作簡報」、「展示方案」或「數據說明」時使用。

## 📏 視覺動線 (Visual Path)
*   **單一焦點**：每張投影片只傳達一個核心資訊 (One slide, one idea)。
*   **Z 字型閱讀**：利用排版引導觀眾視線從左上至右下流動。
*   **空白美學**：保留充足的留白 (White space)，避免頁面過於擁擠。

## 📊 數據可視化
*   **重點強調**：在圖表中直接標示出關鍵數據，而非由觀眾自行解讀。
*   **配色統一**：圖表的顏色應與投影片的設計系統保持一致。
*   **簡化訊息**：移除不必要的背景網格與多餘的標註。

## ✨ 精品感細節
*   **微動畫**：使用平滑的平移或淡入效果，避免誇張的翻頁特效。
*   **高質感圖片**：僅使用高解析度的情境圖，嚴禁拉伸圖片。
*   **對齊紀律**：確保所有元素都處於嚴謹的對齊格點上。

## ✅ 專業檢核
*   [ ] 字體大小是否足以讓最後一排觀眾看清？
*   [ ] 每頁文字是否少於 6 行？
*   [ ] 配色是否具備足夠的品牌辨識度？
